HTML 1

  • URL=Uniform Resource Locator=location on the internet
  • HTML=Hyper Text Markup Language=Language for creating websites
    • first line of code always start by telling it's a HTML file
  • <!DOCTYPE html>
  • Elements
    • individual pieces of a HTML document
  • tags tell where an element begins and ends
  • first element in HTML: entire content of HTML file inside it
  • two branches that split off from there, head and body
  • head is for information that isn't visible in the website
  • title is included in the head therefore a HEAD Element, and is displayed on the top of the page
  • Body is the second HTML Element
  • the things in the body element is the main content containing text, images etc.
  • Paragraph is a BODY Element and is used for making paragraphs
  • Header is a BODY Element and is used to make headers
    • you can also use h2 to create a smaller header and it goes to h6
  • Unordered list is a BODY Element and creates an empty bulletpoint list
  • Ordered List is a BODY Element and creates an empty numbered list
  • to fill in the blank list the list item element is used to create one list item (bullet point)
