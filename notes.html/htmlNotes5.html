Semantic Elements

<header>

  • a container for introductory content
  • there can be one or more heading elements (h1-h6)
  • there can also be a logo or icon

<nav>

  • this is a semantic element
  • nav is a container for a set of navigation links

<section>

  • this is a semantic element as well
  • this defines the sections in a document
  • the main content would be inside this container

<footer>

  • this is also a semantic element
  • this is a container for the bottom of the page
  • things like copyright can go inside this container

You can organize your website by using semantic elements
