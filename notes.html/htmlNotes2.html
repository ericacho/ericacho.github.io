Attributes

<br>

  • short for break row
  • inserts a single line break
  • you can add this between two sentences to separate it
  • add two <br> to have another 'enter' space between the sentences

Comments

  • it is code that is ignored by the computer
  • you can use this to 'erase' the code but also still keep it there for reference and recover it.
  • the piece of code will turn green

Attributes: provides additional information about elements

start

  • add 'start=' inside the brackets of the first tag with a number

for instance, if you write 'start="5"' inside the brackets of the first tag for an ordered list, the list will start from the number 5.

start is an attribute and tags can have multiple attributes

reversed

  • reversed attribute specifies that the list order should be descending

you can put in both attributes together to have a list start from a certain number descending
