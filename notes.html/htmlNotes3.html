Images

<img>

  • defines an image in a HTML page

src

  • it is an <img> attribute
  • it specifies the URL of an image, the location of the image on the internet
  • there are two ways to use this attribute
  1. using URL of an image by putting in the src attribute inside the bracket of img
  2. saving the image in the folder containing the HTML code

if you use the first method, and the image is deleted in the internet,the image won't be displayed on your website.

height & width

  • used to resize the image in pixels
