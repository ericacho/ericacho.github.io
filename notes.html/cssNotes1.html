Introduction to CSS

CSS is a website language for styllizing HTML files

<link>

  • links the .html file to .css file
  • it marks the beginning and the end of a hypertext
  • put the clickable phrase inside the <a>

Selector

  • points to the HTML element you want to style

Property

  • stylistic feature you want to change

Value

  • How you want to change its stylistic feature

Common CSS Properties

  1. font-size: used to size the text of a tag/li>
  2. line-height: used to size the height of the text lines
  3. color: used to set the text colour of an element
