Canvas Element (html)

the canvas element is used to draw graphics on a web page

Canvas

Canvas is only a container for the graphics. By default, a canvas is a blank, white rectangle.

ctx

ctx is short for context. The information given after 'ctx =' sets the ctx for the canvas. An example would be to set the context of the canvas to 2D shown in the image below.



window.onload = init

'window.onload = init' means when the window is loaded, run the init function



fillRect

'fillRect' is used for filling in a rectangle with a color. The default color of filling in a rectangel using fillRect is black. The four numbers in the bracket that follow fillRect are the coordinates and size of the rectangel on that canvas. -> (x of top left corner, y of top left corner, width , height)



fillStyle

'fillStyle' specifies the color, shape, gradient of the rectangle.

UNIT

'UNIT' is set as a certain number and you can put in UNIT*number so that when you want to make your image as a whole larger or smaller, you can do that by changing your UNIT not each coordinate of each rectangle in your image.
